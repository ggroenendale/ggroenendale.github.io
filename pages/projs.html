    Data Management - Final Project

    Complete

    Data management was a class that went into detail on the functions of a Microsoft SQL Server database as well as how those functions could help a business manage their data more efficiently. This semester long project required us to create an operational database for a fictitious organization. The database would have to meet several requirements and be a database that produces value for the organization. I chose to make a database based around a project management office doing primarily construction. This document details my final findings and some of my operational queries and procedures.
